"Continuous Control" demonstrates both user control and parameter automation. When the user starts the instrument, a frequency-modulation oscillator is turned on. Amplitude and modulation are controlled by the user with UISliders. The modulation index is automated.

Continuous Control

 

"Play Audio File" demonstrates audio playback capabilities. Each time the button is pressed, the audio file plays, but with randomized playback speed.

Play Audio File

 

This is a Frequency-Modulation Oscillator implemented with Swift. It has five parameters controlled via UISliders: frequency, amplitude, carrier multiplier, modulating multiplier, and modulating index.

Swift FM Oscillator

 

"Touch Regions" is a Frequencey-Modulation Oscillator controlleted through gestures along an X/Y axis. Moving a finger along the X axis of the red section controls frequency, and moving along the Y axis controls the carrier multiplier. Moving along the purple section's X-axis controls the modulating multiplier, and moving along the Y-axis controls the modulation index.

Touch Regions
