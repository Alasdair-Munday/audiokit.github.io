This example is part of the AudioKit Examples GitHub repository.

This is a beginner level example, that only uses one instrument and demonstrates both user control and parameter automation. When the user starts the instrument, a frequency-modulation oscillator is turned on. Amplitude and modulation are controlled by the user with UISliders. The modulation index is automated.

The code for implementing the FM oscillator instrument controlled with user and automated parameters is shown below.

{% highlight objective-c %} AKFMOscillator *fmOscillator = [AKFMOscillator oscillator]; fmOscillator.baseFrequency = _frequency; fmOscillator.modulatingMultiplier = _modulation; fmOscillator.modulationIndex = _modIndex; fmOscillator.amplitude = _amplitude; {% endhighlight %}

Mac Version

The OS X and iOS versions of this example share the exact same AudioKit instrument files, but they use different UI elements to connect to the instrument's properties.
