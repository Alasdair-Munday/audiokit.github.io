Notes are "things" created by an instrument. Usually these "things" are sounds, but a note is not limited to being a sound.

Note Properties

A note can have properties of its own, distinct from the properties of the instrument that it is created on. As an example, a guitar can create many notes, each with its own frequency.

Defining Notes for an Instrument

It is common practice to include a note class definition with the file containing the instrument class. For example, file interface and implementation files defining the tambourine class "Tambourine" may contain the "TambourineNote" class as well. This is handy since you only have to import the one interface file instead of one for the instrument and a second for the note.
