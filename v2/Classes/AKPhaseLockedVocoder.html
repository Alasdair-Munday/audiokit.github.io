Overview

Phase-locked vocoder processing. More detailed description from http://www.csounds.com/manual/html/mincer.html

Instance Methods

initWithTable:time:scaledPitch:amplitude:

Instantiates the phase locked vocoder
{% highlight objective-c %}- (instancetype)initWithTable:(AKControl *)table time:(AKAudio *)time scaledPitch:(AKControl *)scaledPitch amplitude:(AKControl *)amplitude{% endhighlight %}
  • table: Source signal table. Only mono audio files work.
  • time: Time position of current audio sample in secs. Table reading wraps around the ends of the table.
  • scaledPitch: Grain pitch scaling (1=normal pitch, < 1 lower, > 1 higher; negative, backwards)
  • amplitude: Amplitude scaling

setOptionalDecimation:

Set an optional decimation
{% highlight objective-c %}- (void)setOptionalDecimation:(AKConstant *)decimation{% endhighlight %}
  • decimation: Defaults to 4 (meaning hopsize = fftsize/4)

setOptionalSizeOfFFT:

Set an optional size of fft
{% highlight objective-c %}- (void)setOptionalSizeOfFFT:(AKConstant *)sizeOfFFT{% endhighlight %}
  • sizeOfFFT: FFT size (power-of-two), defaults to 2048.
