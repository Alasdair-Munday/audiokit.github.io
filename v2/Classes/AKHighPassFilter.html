AKHighPassFilter.m

Overview

A first-order recursive high-pass filter with variable frequency response. A complement to the AKLowPassFilter.

Properties

cutoffFrequency

The response curve's half-power point, in Hertz. Half power is defined as peak power / root 2. [Default Value: 4000]
