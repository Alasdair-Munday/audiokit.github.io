initWithGrainWaveform:frequency:windowWaveform:duration:density:maximumOverlappingGrains:frequencyVariation:frequencyVariationDistribution:phase:startPhaseVariation:prpow:

Instantiates the granular synthesizer with all values

- (instancetype)initWithGrainWaveform:(AKTable *)grainWaveform frequency:(AKParameter *)frequency windowWaveform:(AKTable *)windowWaveform duration:(AKParameter *)duration density:(AKParameter *)density maximumOverlappingGrains:(AKConstant *)maximumOverlappingGrains frequencyVariation:(AKParameter *)frequencyVariation frequencyVariationDistribution:(AKParameter *)frequencyVariationDistribution phase:(AKParameter *)phase startPhaseVariation:(AKParameter *)startPhaseVariation prpow:(AKParameter *)prpow

Parameters

grainWaveform

Table contain grain waveform. Updated at Control-rate. [Default Value: ]

frequency

Grain frequency in Hz Updated at Control-rate. [Default Value: ]

windowWaveform

Table containing window waveform. [Default Value: AKWindowTypeHamming]

duration

Grain duration in seconds. It also controls the duration of already active grains (actually the speed at which the window function is read). Updated at Control-rate. [Default Value: 0.2]

density

Number of grains per second. Updated at Control-rate. [Default Value: 200]

maximumOverlappingGrains

The maximum number of overlapping grains. [Default Value: 200]

frequencyVariation

Random bipolar variation in grain frequency in Hz. Updated at Control-rate. [Default Value: 0]

frequencyVariationDistribution

This value controls the distribution of grain frequency variation. If this is positive, the random distribution (x is in the range -1 to 1) isabs(x) ^ ((1 / parameter) - 1) For negative parameter values, it is (1 - abs(x)) ^ ((-1 / parameter) - 1) Setting this parameter to -1, 0, or 1 will result in uniform distribution (this is also faster to calculate). Updated at Control-rate. [Default Value: 0]

phase

Grain phase. This is the location in the grain waveform table, expressed as a fraction (between 0 to 1) of the table length. Updated at Control-rate. [Default Value: 0.5]

startPhaseVariation

Random variation (bipolar) in start phase Updated at Control-rate. [Default Value: 0.5]

prpow

Distribution of random phase variation. Updated at Control-rate. [Default Value: 0]
