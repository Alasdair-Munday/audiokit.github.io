AKFilteredFFT.m

Overview

Multiply amplitudes of an f-signal by those of a second f-signal, with dynamic scaling. More detailed description from http://www.csounds.com/manual/html/pvsfilter.html

Properties

gain

Amplitude scaling [Default Value: 1.0]

  • input: Input AKFSignal, from which amplitudes and frequencies are taken.
  • amplitude: Amplitude AKFSignal, the amplitudes of which are multiplied by the input signal.
  • depth: Controls the depth of filtering of the input signal by the amplitude signal.
