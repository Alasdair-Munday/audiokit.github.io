Overview

Generate control signal consisting of linear or exponential segments.

Generate control signal consisting of linear segments delimited by two or more specified points. The entire envelope is looped at a rate defined by frequency. Each parameter can be varied as controls.

addValue:afterDuration:concavity:

Adds another segment.

- (void)addValue:(AKParameter *)value afterDuration:(AKParameter *)duration concavity:(AKParameter *)concavity

Parameters

value

Value after nextSegmentDuration seconds.

duration

Dimensionless duration - sum all durations will be rescaled to the period given by the inverse of the frequency of the loop

concavity

A value greater than zero is an initially slower change and less than zero is a quicker intial change. Use zero for a linear change.
