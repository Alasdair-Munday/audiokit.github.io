AKControlSegmentArray Class Reference

Inherits from AKControl : AKParameter : NSObject
Declared in AKControlSegmentArray.h
AKControlSegmentArray.m

Overview

Trace a series of line segments between specified points. The transition between these points can either be linear, by default, or exponential, by sending invoking the useExponentialSegments method.

Tasks

Other Methods

  • – initWithStartValue:toNextValue:afterDuration:

    Creates the AKSegmentArray and populates it with the minimum information. Use addValue:afterDuration to add more segments to the array. An optional release segment can be added with addReleaseToFinalValue:afterDuration.
  • – addValue:afterDuration:
