Overview

Trace a series of exponential segments between specified points.

Tasks

  • – initWithInitialValue:targetValue:afterDuration:

    Creates the exponential segment array and populates it with the minimum information. Use addValue:afterDuration to add more segments to the array. An optional release segment can be added with addReleaseToFinalValue:afterDuration.
  • – addValue:afterDuration:

    Adds another segment.
