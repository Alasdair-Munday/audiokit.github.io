Overview

A first-order recursive low-pass filter with variable frequency response. More detailed description from http://www.csounds.com/manual/html/tone.html

Properties

halfPowerPoint

The response curve's half-power point, in Hertz. Half power is defined as peak power / root 2. [Default Value: 1000]
