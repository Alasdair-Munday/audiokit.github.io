betaDistributionWithAlpha:beta:

Create a random distribution table of the Beta (positive numbers only) type. If both alpha and beta equal one we have uniform distribution. If both alpha and beta are greater than one we have a sort of Gaussian distribution.

+ (instancetype)betaDistributionWithAlpha:(float)alpha beta:(float)beta

Parameters

alpha

If alpha is smaller than one, smaller values favor values near 0.

beta

If beta is smaller than one, smaller values favor values near level.
