Overview

Applies a straight line rise and decay pattern to an input amp signal.

Rise modifications are applied for the first irise seconds, and decay from time totalDuration - decayTime. If these periods are separated in time there will be a steady state during which amp will be unmodified. If linen rise and decay periods overlap then both modifications will be in effect for that time. If the overall duration is exceeded in performance, the final decay will continue on in the same direction, going negative.

Instance Methods

initWithRiseTime:totalDuration:decayTime:amplitude:

Creates a straight line rise and decay patter to an input signal.

- (instancetype)initWithRiseTime:(AKConstant *)riseTime totalDuration:(AKConstant *)totalDuration decayTime:(AKConstant *)decayTime amplitude:(AKParameter *)amplitude

Parameters

riseTime

Rise time in seconds. A zero or negative value signifies no rise modification.

totalDuration

Overall duration in seconds. A zero or negative value will cause initialization to be skipped.

decayTime

Decay time in seconds. Zero means no decay. If it is greater than the total duration, it will cause a truncated decay.

amplitude

Amplitude to rise to and decay from.
