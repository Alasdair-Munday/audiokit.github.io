Installation

We include a variety of sample projects alongside the toolkit so that you can get started right away. Here's how to get the complete package.

Installing using Git

Installing from command line (defaults to "develop" branch):

{% highlight bash %} $ git clone git@github.com:audiokit/AudioKit.git {% endhighlight %}

Archives are available:

  • Master Branch (24.4MB zip) - most stable
  • Development Branch (24.4MB zip) - latest and greatest

Installing using CocoaPods

If you are creating an iOS project using Objective-C, you can use CocoaPods to install AudioKit. In your project, update your Podfile:

{% highlight ruby %} pod 'AudioKit', '~> 1.3' {% endhighlight %} Then execute: {% highlight bash %} $ pod install {% endhighlight %}

Installing Xcode Documentation

Code completion, pop-ups, and sidebar information are automatically created in Xcode, but if you would like Apple-style API reference documentation, download the docset:

Un-tar it and place it in the Shared Documentation folder:

{% highlight bash %} tar -xf ~/Downloads/io.audiokit.AudioKit.docset.tar.gz -C ~/Library/Developer/Shared/Documentation/DocSets/ {% endhighlight %}
