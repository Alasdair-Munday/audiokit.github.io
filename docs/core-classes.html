ClassDescription
AKFoundation This header file contains all the includes you might need for any AudioKit file
AKManager The manager controls all the interaction with the audio engine.
AKOrchestra This is the set of instruments available to play.
AKInstrument This is one sound generator or processor which has its own controllable pproperties.
AKNote This is an instance of a instrument sound that may have separate properties to the instrument (for instance a guitar note has its own frequency from other simultaneous guitar notes).
AKMidi The object that handles midi in and out information.
Handling multiple notes or instructions is done through creation of events which can be combined into sequences:
ClassDescription
AKPhrase A collection of notes and start times that can be played by an instrument.
AKEvent An event that can be created and triggered at any time.
AKSequence A series of events.

Parameters, properties, and operations are also core classes, but they have many subclassed types, so they are each presented separately in this documentation.
