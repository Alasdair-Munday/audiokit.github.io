ClassDescription
AKAdditiveCosineTable Generates an additive set of cosine partials.
AKArrayTable Constructs a function table out of an NSArray.
AKCompositeWaveformsFromSines Generates composite waveforms made up of weighted sums of sinusoids.
AKExponentialCurvesTable Constructs functions from segments of exponential curves.
AKFTable Generic AK Function Table definiton.
AKLineSegmentTable Constructs functions from segments of straight lines.
AKRandomDistributionTable Generates tables of different random distributions.
AKSineTable Generates composite waveforms made up of weighted sums of simple sinusoids.
AKSoundFileTable Transfers data from a soundfile into a function table.
AKWindowsTable Generates functions of different windows.
