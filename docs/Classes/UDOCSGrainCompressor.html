Overview

Stereo compressor from Boulanger Labs' csGrain application. Stereo audio input and output.

The threshold will normally be 0 or less, but if higher the threshold will begin removing low-level signal energy such as background noise.

A compression ratio value of 2 will advance the output just one decibel for every input gain of two; 3 will advance just one in three; 20 just one in twenty, etc.

Inverse ratios will cause signal expansion: .5 gives two for one, .25 four for one, etc. The value 1 will result in no change.

Parameters

Input to the left and right channels.

The lowest decibel level that will be allowed through.

compressionRatio

The ratio of compression(>1) or exapansion(<1) when the signal level is above the knee.

The attack time in seconds. A typical value is 0.01 seconds.

The release time in seconds. A typical value is 0.1 seconds.
