Instantiates the ranular synthesis texture

- (instancetype)initWithGrainFTable:(AKConstant *)grainFTable windowFTable:(AKConstant *)windowFTable maximumGrainDuration:(AKConstant *)maximumGrainDuration averageGrainDuration:(AKControl *)averageGrainDuration maximumFrequencyDeviation:(AKControl *)maximumFrequencyDeviation grainFrequency:(AKParameter *)grainFrequency maximumAmplitudeDeviation:(AKControl *)maximumAmplitudeDeviation grainAmplitude:(AKParameter *)grainAmplitude grainDensity:(AKParameter *)grainDensity

Parameters

grainFTable

The grain waveform. This can be just a sine wave or a sampled sound.

windowFTable

The amplitude envelope used for the grains.

maximumGrainDuration

Maximum grain duration in seconds.

averageGrainDuration

Average grain duration in seconds.

maximumFrequencyDeviation

Maximum pitch deviation from grainFrequency in Hz.

grainFrequency

To use the original frequency of the input sound, divide the original sample rate of the grain waveform by the length of the grain function table.

maximumAmplitudeDeviation

Maximum amplitude deviation from amplitude. If it is set to zero then there is no random amplitude for each grain.

grainAmplitude

Amplitude of each grain.

grainDensity

Density of grains measured in grains per second. If this is constant then the output is synchronous granular synthesis. If grainDensity has a random element (like added noise), then the result is more like asynchronous granular synthesis.
