Overview

Physical model of the plucked string.

A user can control the pluck point, the pickup point, the filter, and an additional audio signal used to excite the ‘string’. Based on the Karplus-Strong algorithm.

initWithFrequency:pluckPosition:amplitude:samplePosition:reflectionCoefficient:excitationSignal:

Instantiates the plucked string

- (instancetype)initWithFrequency:(AKConstant *)frequency pluckPosition:(AKConstant *)pluckPosition amplitude:(AKControl *)amplitude samplePosition:(AKControl *)samplePosition reflectionCoefficient:(AKControl *)reflectionCoefficient excitationSignal:(AKAudio *)excitationSignal

Parameters

frequency

Frequency of the string

pluckPosition

The point of pluck as a fraction of the way up the string (0 to 1). A pluck point of zero means no initial pluck.

amplitude

Amplitude of note.

samplePosition

Proportion of the way along the string to sample the output.

reflectionCoefficient

The coefficient of reflection, indicating the lossiness and the rate of decay. It must be strictly between 0 and 1 (it will complain about both 0 and 1).

excitationSignal

A signal which excites the string.
