initWithSourceStereoAudio:basePitch:offsetFrequency:feedbackLevel:

Instantiates the pitch shifter.

- (instancetype)initWithSourceStereoAudio:(AKStereoAudio *)sourceStereo basePitch:(AKControl *)basePitch offsetFrequency:(AKControl *)fineTuningOffsetFrequency feedbackLevel:(AKControl *)feedbackLevel

Parameters

sourceStereo

Input to the left and right channels.

basePitch

The pitch to shift by in pitch notation.

fineTuningOffsetFrequency

Frequency in Hz that will be added to the converted pitch frequency (a negative will detune).

feedbackLevel

Typically a value from 0.0 (no feedback to 1.0 (100% feedback).

Return Value

An instance of the pitch shifter.
