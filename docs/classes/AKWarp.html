initWithSourceSignal:scalingRatio:shift:

Instantiates the warp

- (instancetype)initWithSourceSignal:(AKFSignal *)sourceSignal scalingRatio:(AKControl *)scalingRatio shift:(AKControl *)shift

Parameters

sourceSignal

Input Stream

scalingRatio

Spectral envelope scaling ratio. Values > 1 stretch the envelope and < 1 compress it.

shift

Spectral envelope shift, values > 0 shift the envelope linearly upwards and values < 1 shift it downwards.
