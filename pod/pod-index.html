Installing using CocoaPods

If you are creating an iOS project using Objective-C, you can use CocoaPods to install AudioKit. In your project, update your Podfile:

{% highlight ruby %} pod 'AudioKit', '~> 2.1.1' {% endhighlight %} Then execute: {% highlight bash %} $ pod install {% endhighlight %}

Version 2.2 has not been able to pass validation so no CocoaPod is available for it yet. In the meantime you can use the github repo directly in your Podfile:

{% highlight bash %} pod 'AudioKit', :git => 'https://github.com/audiokit/AudioKit' {% endhighlight %}
