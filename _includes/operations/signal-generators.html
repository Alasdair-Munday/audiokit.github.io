Envelopes

ClassDescription
AKADSREnvelope Calculates the classical ADSR envelope using exponential segments.
AKLine Creates an infinitely long line that connects a starting to an ending point over the given time duration.
AKLinearADSREnvelope Calculates the classical ADSR envelope using linear segments.
AKLinearEnvelope Applies a straight line rise and decay pattern to an input amp signal.

ClassDescription
AKGranularSynthesisTexture Generates granular synthesis textures.
AKSinusoidBursts Produces sinusoid bursts useful for formant and granular synthesis.

Musical Controls

ClassDescription
AKPortamento Applies portamento to a step-valued control signal.

Oscillators

ClassDescription
AKFMOscillator Basic frequency modulated oscillator with linear interpolation.

AKOscillator A simple oscillator with linear interpolation.
AKPhasor A normalized moving phase value.
AKVCOscillator Implementation of a band-limited oscillator using pre-calculated tables. Meant to model vintage analog synthesizers.
