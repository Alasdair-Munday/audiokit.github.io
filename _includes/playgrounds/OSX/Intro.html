Introduction

Making sure things work on your system by playing back audio with AKAudioPlayer

{% highlight ruby %}import XCPlayground import AudioKit {% endhighlight %}

All AudioKit powered apps need a reference to the AKManager

{% highlight ruby %}let audiokit = AKManager.sharedInstance {% endhighlight %}

Try Changing "mixloop" to "drumloop"

{% highlight ruby %}let bundle = NSBundle.mainBundle() let file = bundle.pathForResource("mixloop", ofType: "wav") let player = AKAudioPlayer(file!) {% endhighlight %}

Next set AudioKit's main audio output to be this player

{% highlight ruby %}audiokit.audioOutput = player {% endhighlight %}

Start up AudioKit to connect all nodes to the system

{% highlight ruby %}audiokit.start() {% endhighlight %}

Start the player's audio playback

{% highlight ruby %}player.play() {% endhighlight %}

Because we need to keep this playground running to playback the audio, we add the following line at the bottom to most playgrounds (or in some cases, we can set up a never ending loop)

{% highlight ruby %}XCPlaygroundPage.currentPage.needsIndefiniteExecution = true {% endhighlight %}
