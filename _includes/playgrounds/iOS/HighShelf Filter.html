AKHighShelfFilter

A high-pass filter takes an audio signal as an input, and cuts out the low-frequency components of the audio signal, allowing for the higher frequency components to "pass through" the filter.

{% highlight ruby %}import XCPlayground import AudioKit let audiokit = AKManager.sharedInstance let bundle = NSBundle.mainBundle() let file = bundle.pathForResource("mixloop", ofType: "wav") var player = AKAudioPlayer(file!) player.looping = true var highShelfFilter = AKHighShelfFilter(player) var h2 = AKHighShelfFilter(highShelfFilter) {% endhighlight %}

Set the parameters of the High-Shelf Filter here

{% highlight ruby %}highShelfFilter.cutOffFrequency = 2000 // Hz highShelfFilter.gain = 40 // dB audiokit.audioOutput = h2 audiokit.start() player.play() {% endhighlight %}

Toggle processing on every loop

{% highlight ruby %}AKPlaygroundLoop(every: 3.428) { () -> () in if highShelfFilter.isBypassed { highShelfFilter.start() } else { highShelfFilter.bypass() } highShelfFilter.isBypassed ? "Bypassed" : "Processing" // Open Quicklook for this } XCPlaygroundPage.currentPage.needsIndefiniteExecution = true {% endhighlight %}
