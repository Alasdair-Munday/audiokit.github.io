--- title: Audio Kit Examples - Swift Convolution header: Swift Convolution permalink: /examples/SwiftConvolution/ layout: examples ---

The first part is to load the loop and the impulse response files:

{% highlight objective-c %} let file = String(NSBundle.mainBundle().pathForResource("808loop", ofType: "wav")!) let loop = AKFileInput(filename: file) connect(loop) let dish = String(NSBundle.mainBundle().pathForResource("dish", ofType: "wav")!) let well = String(NSBundle.mainBundle().pathForResource("Stairwell", ofType: "wav")!) {% endhighlight %}

Perform the convolutions:

{% highlight objective-c %} let dishConv = AKConvolution(audioSource: loop.leftOutput, impulseResponseFile: dish) connect(dishConv) let wellConv = AKConvolution(audioSource: loop.rightOutput, impulseResponseFile: well) connect(wellConv) {% endhighlight %}

and mix the results:

{% highlight objective-c %} let balance = AKMixedAudio(signal1: dishConv, signal2: wellConv, balance: dishWellBalance) connect(balance) let dryWet = AKMixedAudio(signal1: loop.leftOutput, signal2: balance, balance: dryWetBalance) connect(dryWet) {% endhighlight %}
