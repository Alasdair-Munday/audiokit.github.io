This example demonstrates a simpler harmonzier, with the input signal being the device microphone. The code below (from "HarmonizerInstrument.swift") takes the audio input from the device microphone and applies a window to the signal.

{% highlight ruby %} let microphone: AKAudioInput = AKAudioInput () connect (microphone) let fsig1 = AKFSignalFromMonoAudio( audioSource: microphone, fftSize: 2048.ak, overlap: 256.ak, windowType: kVonHannWindow, windowFilterSize: 2048.ak) connect(fsig1) {% endhighlight %}

Notice that float and integers can't be directly used, they need to be converted into AudioKit constant parameters, which is simple done by append .ak to the end of the number. This analagous to the Objective-C function akp().

The input signal is then altered with two instances of "AKScaledFSignal", which scales the frequency of a phase vocoder stream, which results in a pitch shift.

{% highlight ruby %} AKScaledFSignal *fsig2; let fsig2 = AKScaledFSignal(input: fsig1, frequencyRatio: 2.ak, formantRetainMethod: kFormantRetainMethodLifteredCepstrum, amplitudeRatio: nil, cepstrumCoefficients: nil) connect(fsig2) let fsig3 = AKScaledFSignal(input: fsig1, frequencyRatio: 2.ak, formantRetainMethod: kFormantRetainMethodLifteredCepstrum, amplitudeRatio: nil, cepstrumCoefficients: nil) connect (fsig3) {% endhighlight %}

These phase vocoder signals are then mixed with an instance of AKFSignalMix:

{% highlight ruby %} let fsig4 = AKFSignalMix(input1: fsig2, input2: fsig3) connect(fsig4) {% endhighlight %}

Finally, the audio is resynthesized from the phase vocoder data using an instance of "AKAudioFromFSignal":

{% highlight ruby %} let a1 = AKAudioFromFSignal(source: fsig4) connect(a1) {% endhighlight %}
