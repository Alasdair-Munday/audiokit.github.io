Continuous Control

This demonstrates both user control and parameter automation. When the user starts the instrument, a frequency-modulation oscillator is turned on. Amplitude and modulation are controlled by the user with UISliders. The modulation index is automated.

The code for implementing the FM oscillator instrument controlled with user and automated parameters is shown below.

{% highlight objective-c %} AKSineTable *sineTable = [[AKSineTable alloc] init]; [self addFTable:sineTable]; AKFMOscillator *fmOscil; fmOscil = [[AKFMOscillator alloc] initWithFTable:sineTable baseFrequency:_frequency carrierMultiplier:akp(1) modulatingMultiplier:_modulation modulationIndex:_modIndex amplitude:_amplitude]; [self connect:fmOscil]; {% endhighlight %}
