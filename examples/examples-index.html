Included with AudioKit are numerous example projects. There are examples for both Mac and iOS and they are written in both Objective-C and Swift.

Example App Videos

These videos give you a sense of what can be achieved quickly using AudioKit.

Keyboard and Swift Keyboard

This examples demonstrates how a simple keyboard can be created with UIButtons for keys and UISliders to control instrument properties and a global reverb effect. more →


Touch Regions

A Frequency-Modulation Oscillator is controlleted through gestures along an X/Y axis. Moving a finger along the X axis of the red section controls frequency, and moving along the Y axis controls the carrier multiplier. Moving along the purple section's X-axis controls the modulating multiplier, and moving along the Y-axis controls the modulation index. more →


Song Library Player

Processing is not limited to assets you include in your app. You have full access to the user's iTunes Library and can import songs and process them as you wish. The included example demonstrates choosing a song by artist name and album title and allows you to create and mix a reverb effect on that song. more →

Continuous Control

This example demonstrates user control of modulation and amplitude using sliders and parameter automation of modulation index. more →

Convolution and Swift Convolution

Demonstrates convolving a drum loop with impulses of a dish or a stairwell resulting an interesting reverb effect that can be mixed. more →
