Included with AudioKit are numerous example projects. There are examples for both Mac and iOS and they are written in both Objective-C and Swift.

In addition to the included examples, open-source projects that use AudioKit will also be highlighted here.

Hello World

We've recently included a Hello World app for iOS or OSX into AudioKit's embedded Examples directory. Start here if you are new to AudioKit.

AudioKit Demo

This is an app with three tabs that demonstrate the synthesis, processing, and analysis capabilities of AudioKit. AudioKit Demo is a great example to look at once you've done HelloWorld and the tutorials.

Example App Videos

These videos give you a sense of what can be achieved quickly using AudioKit.

Keyboard and Swift Keyboard

This examples demonstrates how a simple keyboard can be created with UIButtons for keys and UISliders to control instrument properties and a global reverb effect. more →

Touch Regions

A Frequency-Modulation Oscillator is controlled through gestures along an X/Y axis. Moving a finger along the X axis of the red section controls frequency, and moving along the Y axis controls the carrier multiplier. Moving along the purple section's X-axis controls the modulating multiplier, and moving along the Y-axis controls the modulation index. more →

Effects Processor Demo

Processing is not limited to assets you include in your app. You have full access to the user's iTunes Library and can import songs and process them as you wish. The included example demonstrates choosing a song by artist name and album title and allows you to create and mix a growing library of effects on that song. more →


Continuous Control

This example demonstrates user control of modulation and amplitude using sliders and parameter automation of modulation index. more →

Convolution and Swift Convolution

Demonstrates convolving a drum loop with impulses of a dish or a stairwell resulting an interesting reverb effect that can be mixed. more →



External Examples

These examples are a bit more complex than the included examples, but they are worth exploring as you learn AudioKit

Space Cannon

This example demonstrates using AudioKit to create a procedurally generated soundtrack for a game written using Apple's SpriteKit. It utitilized the "Conductor" class pattern to orchestrate the sounds based on the game mechanics.

Screencast: AudioKit for Game Audio: Space Cannon
Repository: http://github.com/audiokit/SpaceCannon/

PendulaTone

Simon Gladman's second use of AudioKit in a project demonstrates using program logic in Swift to generate animation and sounds that trigger on events.

Blog: The Sound of Pendulum Waves with Swift and AudioKit
Repository: https://github.com/FlexMonkey/PendulaTone

AudioSynthesis
