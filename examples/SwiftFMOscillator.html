This is a Frequency-Modulation Oscillator implemented with Swift. It has five parameters controlled via UISliders: frequency, amplitude, carrier multiplier, modulating multiplier, and modulating index.

The code for creating the FM oscillator instrument is shown below.

{% highlight ruby %} override init() { super.init() addProperty(frequency) addProperty(amplitude) addProperty(carrierMultiplier) addProperty(modulatingMultiplier) addProperty(modulationIndex) let sine = AKSineTable() addFTable(sine) let fmOscillator = AKFMOscillator( FTable: sine, baseFrequency: frequency, carrierMultiplier: carrierMultiplier, modulatingMultiplier: modulatingMultiplier, modulationIndex: modulationIndex, amplitude: amplitude ) {% endhighlight %}
