Sequences

"Sequences" demonstrates three methods of creating a series of notes: creating a separate note in the sequence, altering note properties, and updating instrument properties.

The code below (from 'SequencesConductor.h') controls whether a note is on or off, and also changes the instrument's modulation value.

{% highlight objective-c %} AKEvent *noteOn = [[AKEvent alloc] initWithNote:note]; [sequence addEvent:noteOn]; for (int i = 0; i <=12 ; i++) { AKEvent *update= [[AKEvent alloc] initWithBlock:^{ instrument.modulation.value = pow(2.0f,(float)i/12); }]; [sequence addEvent:update atTime:duration*i]; } {% endhighlight %}
