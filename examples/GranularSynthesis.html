Here we demonstrate granular synthesis, with two audio files as the sources. The parameters that can be altered are: {% highlight objective-c %} @property (nonatomic, strong) AKInstrumentProperty *mix; @property (nonatomic, strong) AKInstrumentProperty *frequency; @property (nonatomic, strong) AKInstrumentProperty *duration; @property (nonatomic, strong) AKInstrumentProperty *density; @property (nonatomic, strong) AKInstrumentProperty *frequencyVariation; @property (nonatomic, strong) AKInstrumentProperty *frequencyVariationDistribution; {% endhighlight %} where mix is simply the balance betwen the two inputs, but the others are characteristic of AKGranularSynthesizer.

The we create two instances of AKGranularSynthesizer, where the first step is to load an audio file into a sound file table.

{% highlight objective-c %} NSString *file = [[NSBundle mainBundle] pathForResource:@"PianoBassDrumLoop" ofType:@"wav"]; AKSoundFile *soundFile; soundFile = [[AKSoundFile alloc] initWithFilename:file]; soundFile.size = 16384; [self addFunctionTable:soundFile]; {% endhighlight %}

And then create the synthesizer:

{% highlight objective-c %} AKGranularSynthesizer *synth; synth = [[AKGranularSynthesizer alloc] initWithGrainWaveform:soundFile frequency:_frequency]; synth.duration = _duration; synth.density = _density; synth.frequencyVariation = _frequencyVariation; synth.frequencyVariationDistribution = _frequencyVariationDistribution; [self connect:synth]; {% endhighlight %}
