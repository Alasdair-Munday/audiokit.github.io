This example is part of the AudioKit Examples GitHub repository.

This project demonstrates applying a chain of effects to an input aduio stream. Audio can come from the user's iTunes library or a set of included sampled audio files. The effects include variable delay, a Moog Ladder filter, and reverb, but we are adding to that list regularly.
