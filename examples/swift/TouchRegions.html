This example is part of the AudioKit Examples GitHub repository.

A Frequency-Modulation Oscillator is controlleted through gestures along an X/Y axis. Moving a finger along the X axis of the red section controls frequency, and moving along the Y axis controls the carrier multiplier. Moving along the purple section's X-axis controls the modulating multiplier, and moving along the Y-axis controls the modulation index.

The code below shows an FM oscillator with four parameters of control. Unlike the other example examples, the parameters for this instrument are controlled based on the position of touches on an X/Y axis.

{% highlight objective-c %} let fmOscillator = AKFMOscillator( waveform: AKTable.standardSineWave(), baseFrequency: frequency, carrierMultiplier: carrierMultiplier, modulatingMultiplier: modulatingMultiplier, modulationIndex: modulationIndex, amplitude: amplitude ) {% endhighlight %}
