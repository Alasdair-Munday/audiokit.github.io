The Global Effects example illustrates the use of reverb as a global effect on an audio signal.

The code below, from "EffectsProcessor.h", creates a reverb effect that modifies an input signal.The input signal is created in "ToneGenerator.m".

{% highlight objective-c %} AKReverb * reverb = [[AKReverb alloc] initWithAudioSource:audioSource feedbackLevel:akp(0.8) cutoffFrequency:akp(12000)]; [self connect:reverb]; {% endhighlight %}

The code below sends the reverb-affected audio signal to the output.

{% highlight objective-c %} AKAudioOutput *audio; audio = [[AKAudioOutput alloc] initWithSourceStereoAudio:[reverb scaledBy:akp(0.2)] ]; [self connect:audio]; {% endhighlight %}
