We have included a lot of useful utilities within AudioKit to help you get a project running quickly. None of these utilities are required by AudioKit for it to run (ie. none of them are imported into "AKFoundation.h") but you can pick and choose which features you need.

ClassDescription
AKSampler AKSampler is a simple multi-track capable recorder and playback system.
AKTools AKTools is our large suite of helper functions.

Instruments

Amplifiers
ClassDescription
Amplifier A monophonic amplification system with amplitude control.
StereoAmplifier A stereo amplification system with amplitude control.
Analyzers
ClassDescription
AKAudioAnalyzer An instrument that has two properties that detect the amplitude and frequency of an audio source signal.
Emulations
ClassDescription
BambooSticks An instrument that wraps the bamboo sticks physical model
Mandolin An instrument that wraps the mandolin physical model
Marimba An instrument that wraps the marimba physical model
PluckedString An instrument that wraps the plucked string physical model
Sleighbells An instrument that wraps the sleighbells physical model
Stick An instrument that wraps the stick physical model
StruckMetalBar An instrument that wraps the struck metal bar physical model
Tambourine An instrument that wraps the tambourine physical model
Vibraphone An instrument that wraps the vibraphone physical model
Processors
ClassDescription
ReverbProcessor A reverb processor and end point (outputs to the DAC)
Synthesizers
ClassDescription
FMOscillatorInstrument A synth that uses FM Synthesis to generate sounds, with frequency and amplitude defined
VCOscillatorInstrument An instrument for the Voltage-controlled oscillator operation

Plots

ClassDescription
AKAudioInputPlot Plots the incoming audio source signal, usually the microphone
AKAudioOutputPlot Plot the raw samples of the audio output to the DAC
AKFFTPlot Plots the FFT of the audio output
AKFloatPlot Plots the floating point value given a minimum and maximum
AKInstrumentPropertyPlot Plot of the given instrument property
AKStereoOutputPlot Plot the raw samples of the audio output to the DAC as left and right signals
AKTablePlot Plots the values of the given table

User Interface Elements

ClassDescription
AKPropertyLabel Ties a label to the value of a property
AKPropertySlider A slider that sets the value of a property (within the property's bounds)
LevelMeter A level meter that can be drawn from any property
